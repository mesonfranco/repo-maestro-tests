**FlipaClip - Smoke Test**

It covers the following cases:

**App Startup**

- Pass through the Welcome flow on all its steps.

**App settings**

- Change stage layout.

- Override movie names.

- Turn off Splash animation.

**Accounts**

- Log in/out YouTube account.

- Log in/out FlipaClip account.

**Project management**

- Create, duplicate, delete and search projects.

**Movies management**

- Create a project with 12 empty frames using the FlipaClip layout 1.

- Create the movie validating the Duration, FPS, and total frames match the movie details.

- Share the movie on YouTube.

- Create a project with 12 empty frames using the FlipaClip layout 2.

- Delete movies.

**API**

- Load Audio Library.

- Load Discover section.

- Load Watch section.
